# Desktop-kierros Runko

Tama on nopein tapa kayda koko sivusto lapi desktopilla ennen launchia.

Kayta samaa runkoa jokaisella sivulla.

## Testijarjestys

Kay sivut tassa jarjestyksessa:
- `/`
- `/service`
- `/process`
- `/cases`
- `/about-us`
- `/contact`
- `/book`
- `/privacy`
- `/terms`

## Tarkista jokaisella sivulla

1. Hero
- [ ] otsikko nayttaa hyvalta
- [ ] ingressi ei tunnu liian pitkältä
- [ ] CTA:t ovat heti ymmarrettavat
- [ ] hero-visual ei riko layoutia

2. Layout
- [ ] mikaan osio ei mene paallekkain
- [ ] rivit ja kortit pysyvat gridissa
- [ ] spacing tuntuu tasaiselta
- [ ] sivu ei tunnu tyhjalta tai liian ahtaalta

3. Interaktiot
- [ ] napit nayttavat klikattavilta
- [ ] hover/focus tuntuu ehjalta
- [ ] linkit vievat oikeaan paikkaan

4. Footer
- [ ] footer asettuu oikein sivun loppuun
- [ ] footerin linkit nayttavat ehjilta
- [ ] kielilinkit nayttavat oikeilta

## Kirjaa loydot nain

Kirjaa jokaisesta ongelmasta vain nama:
- sivu
- kohta
- mita odotit
- mita tapahtui oikeasti

Esimerkki:
- `/service`
- hero
- CTA-rivin pitaisi mahtua siististi yhdelle alueelle
- toinen nappi putoaa rumasti liian alas 1440px leveydella

## Nopea paatos jokaisen sivun lopussa

Valitse yksi:
- [ ] ok
- [ ] pieni fixi
- [ ] selkea bugi

## Kun kierros on valmis

Jos loydot:
- vain pieniä visuaalisia juttuja -> korjataan ne yksi kerrallaan
- useita samoja layout-ongelmia -> korjataan ensin yhteinen layout
- vain yksittaisia sivukohtaisia ongelmia -> korjataan pahin ensin

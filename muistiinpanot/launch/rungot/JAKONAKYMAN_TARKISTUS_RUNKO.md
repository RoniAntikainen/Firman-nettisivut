# Jakonakyman Tarkistus Runko

Tama on nopein tapa tarkistaa, etta linkin esikatselu nayttaa uskottavalta ennen launchia.

## Tarkista ainakin nama sivut

- `/`
- `/service`

Tarvittaessa myos:
- `/cases`

## Katso jokaisella sivulla

1. Otsikko
- [ ] title nayttaa ehjalta
- [ ] title ei katkea oudosti
- [ ] title kuulostaa uskottavalta

2. Kuvaus
- [ ] description nayttaa ehjalta
- [ ] description ei ole liian pitka
- [ ] description tukee klikkausta

3. Kuva
- [ ] kuva nakyy
- [ ] kuva ei ole rajautunut oudosti
- [ ] kuva tuntuu samalta brandilta kuin sivusto

4. Kokonaisfiilis
- [ ] preview ei nayta rikkinäiseltä
- [ ] preview ei nayta demolta
- [ ] preview tuntuu ammattimaiselta

## Missa tarkistaa

Voit tarkistaa esimerkiksi:
- Slack
- WhatsApp
- LinkedIn
- X

Jos et testaa kaikissa, tarkista ainakin yksi paikka jossa linkkipreview oikeasti nakyy.

## Jos loydat virheen

Kirjaa aina:
- sivu
- missa tarkistit
- mita odotit
- mita tapahtui oikeasti

Esimerkki:
- `/service`
- LinkedIn
- kuvan pitaisi nayttaa koko kortissa selkealta
- preview-kuva nayttaa liian geneeriselta eika erotu

## Nopea paatos lopussa

Valitse jokaiselle tarkistetulle sivulle yksi:
- [ ] ok
- [ ] pieni viilaus
- [ ] selkea ongelma

# Mobiilikierros Runko

Tama on nopein tapa kayda koko sivusto lapi mobiilissa ennen launchia.

Kayta samaa runkoa jokaisella sivulla.

## Testijarjestys

Kay sivut tassa jarjestyksessa:
- `/`
- `/service`
- `/process`
- `/cases`
- `/about-us`
- `/contact`
- `/book`
- `/privacy`
- `/terms`

## Tarkista jokaisella sivulla

1. Header
- [ ] mobiilivalikko aukeaa oikein
- [ ] mobiilivalikko sulkeutuu oikein
- [ ] navigaatio tuntuu selkealta
- [ ] kielenvaihto nayttaa oikealta

2. Hero
- [ ] otsikko ei mene rumasti poikki
- [ ] ingressi mahtuu hyvin
- [ ] CTA:t mahtuvat siististi
- [ ] hero-visual ei tunnu raskaalta tai rikkinäiseltä

3. Sisalto
- [ ] kortit eivat pursua ruudun ulkopuolelle
- [ ] teksti pysyy luettavana
- [ ] spacing tuntuu tasapainoiselta
- [ ] mikaan osa ei tunnu liian ahtaalta

4. Interaktiot
- [ ] nappeja on helppo painaa
- [ ] linkit toimivat oikein
- [ ] ankkurit ja lomakelinkit toimivat

5. Footer
- [ ] footer pysyy luettavana
- [ ] linkit mahtuvat siististi
- [ ] kielilinkit nayttavat ehjilta

## Kirjaa loydot nain

Kirjaa jokaisesta ongelmasta vain nama:
- sivu
- leveys tai laite
- kohta
- mita odotit
- mita tapahtui oikeasti

Esimerkki:
- `/contact`
- iPhone 13 / 390px
- hero CTA:t
- nappien pitaisi pinoutua siististi
- toinen nappi jaa liian ahtaaksi ja teksti katkeaa rumasti

## Nopea paatos jokaisen sivun lopussa

Valitse yksi:
- [ ] ok
- [ ] pieni fixi
- [ ] selkea bugi

## Kun kierros on valmis

Jos loydot:
- toistuvan mobiiliongelman usealla sivulla -> korjataan ensin yhteinen layout tai komponentti
- vain yksittaisia ongelmia -> korjataan pahin ensin
- ei ongelmia -> mobiilikierros on valmis
